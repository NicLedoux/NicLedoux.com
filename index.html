Fizz Buzz

My take on the classic Fizz Buzz Javascript test

This classic javascript test console logs numbers 1 through 100, with numbers divisible by 3 and 5 replaced with 'FizzBuzz', numbers divisible by 3 replaced with 'Fizz' and numbers divisible by 5 replaced with 'Buzz'. Copy and paste it into your console to watch it in action! 


//FizzBuzz
for (let i = 1; i <= 100; i++) {
  if (i % 5 === 0 && i % 3 === 0) {
    console.log(`FizzBuzz`);
  } else if (i % 5 === 0) {
    console.log(`Buzz`);
  } else if (i % 3 === 0) {
    console.log(`Fizz`);
  } else {
    console.log(`${i}`);
  }
}
